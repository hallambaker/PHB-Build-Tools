Time

ASN.1 specifies multiple time values, UTCTime and Generalized Time. RFC5280 requires that applications accept either but use UTCTime to encode times before 2050 and Generalized Time thereafter.

 

 

BitObject / OctetObject

Sometimes an ASN.1 object is encoded as a BIT string, sometimes as an OCTET string.

BitObject maps to <OID, BIT String>

OctetObject maps to <OID, [Critical], OCTET STRING>
