The Extension type exposes the following members.

Methods
  NameDescription
OpenFileAppend
Open a new or existing file for append only write access. Permit concurrent reads but not writes.
OpenFileAppendShare
Open a new or existing file for append only write access. Permit concurrent reads and writes.
OpenFileNew
Create a new file for exclusive write access, overwriting any existing file.
OpenFileRead
Open a file for read access allowing other processes to read the file..
OpenFileReadShared
Open a file for read access in shared mode, allowing concurrent reads and writes.
OpenFileWrite
Open an existing file for exclusive write access, or create new file.
OpenTextReader(FileStream)
Create a text reader on a file stream.
OpenTextReader(String)
Create a text reader for a file permitting other processes to perform concurrent reads.
OpenTextWriter(FileStream)
Open a text writer to the specified file stream.
OpenTextWriter(String)
Open a text writer to the specified file in append mode permitting shared reads but not writes.
