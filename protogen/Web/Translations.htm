Furture Extensions

Curerntly the tool provides only limited handling of numeric types, the only such type being supported being 32 bit (signed) integers. Addition of a Float declaration is planned.

Since JSON does not distinguish between numeric types at all, it seems most appropriate to support use of different representations for integers and floats by means of 'hints' to specify the minimum representation size to be used. This would allow Integers to be represented as fixed length 32 or 64 bit values or if appropriate a bignum value.
